## Create a novel
**protip 1**: The `--gpt4` option is recommended.

## Setup

#### Download
```
git clone git@github.com:danmedani/fictions.git
cd fictions
```

#### Install
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Run
```
python main.py --api-key [OPENAI-API-KEY] [--gpt4]
```
